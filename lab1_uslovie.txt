1. Определиться с тематикой сайта, реализовать по данной тематике трёхстраничный html+css-сайт.
2. Обязательно должен состоять не менее чем из 3-х страниц(главная, страница с данными в табличном виде должны быть обязательно). Таблица не менее чем из 15 строк.
3. Главная страница должна содержать меню, раздел с рисунками, видео, вставка аудио.
4. Язык реализации: HTML, CSS.
5. Включить следующие элементы HTML-страницы (как минимум): заголовки, абзацы, списки, раскрывающиеся элементы, гиперссылки, рисунки, таблицы, формы.
6. С помощью CSS реализовать адаптивную верстку(при сужении страницы, меню должно "перестраиваться" в две колонки, остальные элементы должны пропорционно уменьшаться, сохраняя структуру).
7. Реализованный проект выложить на GitHub, препода сделать ревьюером - rmdeuce
8. Не использовать Bootstrap, препроцессоры.
